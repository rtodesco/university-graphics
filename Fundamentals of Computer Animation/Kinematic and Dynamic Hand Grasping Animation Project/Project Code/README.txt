Commands:

system handSys set position x y z
- changes object position

system handSys mass m
- changes object mass

system handSys set size s
- changes object size

system handSys toggle colour
- changes colour of hand from default to type difference, or back

system handSys toggle frame
- changes object from solid to wireframe, or back

simulator handSim set hand x y z
- changes force of hand once object is in it's grasp

simulator handSim toggle gravity
- turns gravity on or off
